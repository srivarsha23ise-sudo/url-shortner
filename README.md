# 🔗 URL Shortener (Django Project)

## 📌 Project Description

This is a URL Shortener web application built using Django.
It allows users to convert long URLs into short links and track click analytics.

---

## 🚀 Features

* 🔗 Shorten long URLs
* 🔄 Redirect to original URL
* ⏱ Link expiration (5 minutes)
* 📊 Click analytics (track number of clicks)
* 🌐 Simple and clean UI

---

## 🛠️ Technologies Used

* Python
* Django
* SQLite
* HTML, CSS

---

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/srivarsha23ise-sudo/url-shortner.git
```

2. Navigate to project folder:

```bash
cd URLShortnerProject
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the server:

```bash
python manage.py runserver
```

5. Open in browser:

```text
http://127.0.0.1:8000/shorten/
```
